# codemod
Common tools when modifying your code with code

[![Build Status](https://travis-ci.org/jahredhope/codemod.svg?branch=master)](https://travis-ci.org/jahredhope/codemod)

Currently exports

- writeFileWithImports,
- writeFile,
- readFile,
- findFilesWith,
- log,
- writeLog,
- camelCase,
- print,
- clean,
- encodeRegex
